# PSD-Layers-Extractor-for-Unity
Tool for Unity at least 6000.3 LTS which extract Photoshop file's layers in sprites 


For installing just add Assets folder in Project folder

In Unity Editor:
Open via Tools -> PSD Extractor

Minimal PSD/PSB reader that extracts layer names, bounds, and merged pixel data.
Supports PSD version 1 (max 30000×30000, up to 56 channels).
Colour modes: RGB (3). Bit depths: 8 bpc.
